# grafiikat/ — vektorigrafiikka

Pudota grafiikat (ikonit, kuviot, koristeet) tähän numeroituna:

```
grafiikka-1.svg
grafiikka-2.svg
grafiikka-3.svg
...
```

- Viittaus: **grafiikka 1**, **grafiikka 2**, …
- Suositeltu muoto: **.svg** (skaalautuva, terävä joka koossa).
- Vain `grafiikka-numero.svg` — ei välilyöntejä, ei ääkkösiä.
