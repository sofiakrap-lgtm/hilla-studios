# demot/ — demojen esikatselukuvat

Pudota tähän kunkin demon **yläosan kuvakaappaus** (nettisivun hero/yläosa).
Tiedostonimet (näitä demot.html jo odottaa):

```
demo-kukkakauppa.webp
demo-kynsitekniikko.webp
demo-rakennusyritys.webp
demo-kakkukauppa.webp
demo-joogastudio.webp
demo-kauneushoitola.webp
```

- Suositeltu muoto: **.webp** (leveä, esim. 1200×500).
- Kunnes kuva on paikallaan, kortissa näkyy "Esikatselukuva tulossa" -paikka.
- "Katso demo →" -napin osoite (href) lisätään, kun itse demosivu on valmis.
